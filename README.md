## Installation

Clone the repository:

```bash
git clone https://github.com/Apoorva-Bairi/apoorva-portfolio.git



```bash

```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
